## 1. 設計理念

### 1.1 核心原則

```
┌──────────────────────────────────────────────────────────────────────────┐
│                         v3.1 設計核心原則                                 │
└──────────────────────────────────────────────────────────────────────────┘

1. 箱體即容器
   • 包裹放入箱體，箱體移動，包裹不動
   • 包裹全程只移動 2 次（進箱體 + 出箱體）
   • 避免卡住、摔落、混淆

2. 雙向運作
   • 收件：外送員 → 大廳站 → AMR → 樓層站 → 住戶
   • 寄件：住戶 → 樓層站 → AMR → 大廳站 → 快遞員
   • 大廳站和樓層站角色可互調

3. 深度最小化
   • 統一深度 60cm
   • 減少佔地空間
   • 適合大廳、走廊安裝

4. 空間最大化利用
   • 投件口/取貨口左右也放箱體
   • AMR對接區左右也放箱體
   • 充分利用升降機可達範圍

5. 模組化設計
   • 升降機模組：大廳/樓層共用
   • 開口模組：投件/取貨共用
   • AMR對接模組：共用
   • 箱體：S/M 兩種尺寸

6. 單升降機 + 雙向皮帶
   • 一個升降機服務左右兩列（大廳站）
   • 皮帶左推或右推，選擇暫存位
   • 減少成本、減少故障點

7. 正開箱體（寬面開）
   • 開口大，放取方便
   • 符合人體工學

8. AMR 簡化
   • AMR 只需承載平台
   • 箱體本身封閉，不需要 AMR 封閉艙體
   • 大幅降低成本

9. 預留擴充
   • L 箱站可獨立擴充
   • 不影響現有系統
```

### 3.4 開門機構

```
┌──────────────────────────────────────────────────────────────────────────┐
│                         開門機構設計                                      │
└──────────────────────────────────────────────────────────────────────────┘

關鍵：箱體本身沒有馬達，由站點控制開門

【開門流程】

    1. 箱體到達開口位置
    2. 站點電磁鎖觸發器對準箱體電磁鎖
    3. 通電 → 電磁鎖解鎖
    4. 彈簧 → 門自動向下翻開
    5. 操作完成後，手動或機構推門關閉
    6. 斷電 → 電磁鎖自動鎖定
```

## 8. 雙向運作流程

### 8.1 收件流程（外送員 → 住戶）

```
┌──────────────────────────────────────────────────────────────────────────┐
│                         收件流程                                          │
└──────────────────────────────────────────────────────────────────────────┘

Step 1: 外送員到達大廳站
        • 掃描包裹條碼
        • 系統確認收件資訊
        • 系統根據包裹大小選擇 S/M 箱體

Step 2: 空箱體送到開口
        • 升降機取出空箱體
        • 下降到開口位置（95cm）
        • 箱體門打開

Step 3: 外送員放入包裹
        • 放入包裹
        • 關門
        • 箱體暫存

Step 4: AMR 運送
        • 系統排程
        • 升降機將箱體送到 AMR 對接位置（35cm）
        • 箱體推入 AMR
        • AMR 運送到目標樓層

Step 5: 箱體送入樓層站
        • AMR 對接樓層站
        • 箱體推入樓層站
        • 升降機接收，箱體暫存

Step 6: 通知住戶
        • 發送 LINE/APP 通知
        • 住戶到樓層站取件

Step 7: 住戶取件
        • 掃碼或刷門禁卡
        • 箱體送到開口
        • 門打開，取出包裹
        • 空箱體等待回收
```

### 8.2 寄件流程（住戶 → 快遞員）

```
┌──────────────────────────────────────────────────────────────────────────┐
│                         寄件流程                                          │
└──────────────────────────────────────────────────────────────────────────┘

Step 1: 住戶預約寄件
        • 在 APP 選擇快遞公司
        • 輸入收件人資訊
        • 取得寄件碼

Step 2: 住戶到樓層站
        • 掃描寄件碼 或 刷門禁卡
        • 選擇「寄件」功能

Step 3: 空箱體送到開口
        • 升降機取出空箱體
        • 下降到開口位置（95cm）
        • 箱體門打開

Step 4: 住戶放入包裹
        • 放入包裹
        • 關門
        • 箱體暫存

Step 5: AMR 運送
        • 系統排程
        • 箱體推入 AMR
        • AMR 運送到大廳站

Step 6: 箱體送入大廳站
        • AMR 對接大廳站
        • 箱體推入大廳站
        • 升降機接收，箱體暫存

Step 7: 通知快遞員
        • 發送通知給快遞公司
        • 快遞員到大廳站取件

Step 8: 快遞員取件
        • 掃描取件碼
        • 箱體送到開口
        • 門打開，取出包裹
        • 空箱體等待下次使用
```
