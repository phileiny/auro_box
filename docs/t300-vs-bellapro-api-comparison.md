# T300 vs BellaPro API 差異比較

> 整理日期：2026-01-02
> 資料來源：pudu_t300_api.md / pudu_bellapro_api.md

---

## 一、總覽

| 項目 | T300 | BellaPro |
|------|------|----------|
| API 總數 | 62 | ~64 |
| 機型定位 | 頂升型配送機器人 | 餐飲服務機器人 |

---

## 二、T300 獨有功能

### 頂升功能 (T300 核心特色)

| 類型 | API | 說明 |
|------|-----|------|
| 回調 | `notifyLiftingTask` | 頂升任務通知 |
| 回調 | `notifyLiftingStatus` | 頂升機構狀態通知 |
| 任務 | 頂升任務 | 下發頂升類任務 |
| 任務 | 頂升任務控制指令 | 頂升過程中的控制指令 |
| 統計 | 頂升統計數據 | 圖表/列表/明細 |

### 電梯聯動

| 類型 | API | 說明 |
|------|-----|------|
| 回調 | `notifyElevatorUtilizeState` | 梯控通知 |
| 回調 | `notifySwitchMap` | 切換地圖通知 |
| 控制 | 電梯內切換當前使用的樓層地圖 | 多樓層場景下的地圖切換 |
| 控制 | 切換地圖 | 手動切換機器人使用的地圖 |

### 其他 T300 獨有

| 類型 | API | 說明 |
|------|-----|------|
| 回調 | `notifyRechargeStatus` | 回充狀態通知 |
| 控制 | 機器人一鍵回充 V2 | 控制機器人返回充電樁 |
| 通用 | 獲取點位分組 | 點位的分組信息 |

---

## 三、BellaPro 獨有功能

### 餐飲服務模式

| 類型 | 功能 | 說明 |
|------|------|------|
| 統計 | 廣告模式 | 圖表/列表數據 |
| 統計 | 領位模式 | 圖表/列表/明細數據 |
| 統計 | 回盤模式 | 圖表/列表/明細數據 |

### 機器人狀態查詢 (更豐富)

| API | 版本 | 說明 |
|-----|------|------|
| 獲取指定機器人狀態 | V1 | 單機查詢 |
| 獲取指定機器人狀態 | V2 | 單機查詢(擴展) |
| 獲取組中機器人狀態 | V1 | 批量查詢 |
| 獲取組中機器人狀態 | V2 | 批量查詢(擴展) |

## 五、關鍵差異總結

| 功能面向 | T300 | BellaPro |
|----------|------|----------|
| 頂升功能 | ✅ 完整支援 | ❌ 無 |
| 電梯聯動 | ✅ 完整支援 | ❌ 無 |
| 一鍵回充 | ✅ V2 API | ❌ 無專用 API |
| 領位/回盤 | ❌ 無 | ✅ 完整支援 |
| 廣告模式 | ❌ 無 | ✅ 有統計 |
| 控制指令數 | 4 個 | 1 個 |

## 六、選型建議

| 機型 | 適用場景 | 核心功能 |
|------|----------|----------|
| **T300** | 物流/倉儲場景 | 頂升貨架、跨樓層配送 |
| **BellaPro** | 餐飲服務場景 | 領位、回盤、廣告播放 |
