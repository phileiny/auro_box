## 一、回調通知 (Callback Notifications)

回調通知用於雲平台主動推送機器人狀態變更至第三方系統。

### 1.1 機器狀態與綁定

| API 名稱 | 中文說明 | 備註 |
|---------|--------|------|
| `robotErrorWarning` | 機器異常故障回調 | 10分鐘延遲 |
| `robotErrorNotice` | 異常實時通知 | 即時推送 |
| `robotBinding` | 機器綁定門店通知 | |
| `robotUnBinding` | 機器解綁門店通知 | |
| `robotActivate` | 機器激活通知 | |
| `robotStatus` | 機器在線/離線狀態變更通知 | |
| `robotEmergencyRecover` | 急停恢復通知 | |
| `notifyRobotStatus` | 機器工作狀態通知 | |

### 1.2 地圖與位置回調

| API 名稱 | 中文說明 | 備註 |
|---------|--------|------|
| `mapElementChange` | 地圖信息（元素點）上傳/變更/刪除 | |
| `notifyRobotPose` | 機器位置上報 | 實時座標 |
| `notifyRobotMoveState` | 機器移動狀態通知 | |
| `notifySwitchMap` | 切換地圖的任務通知 | |

### 1.3 任務執行

| API 名稱 | 中文說明 | 備註 |
|---------|--------|------|
| `notifyCustomCall` | 呼叫狀態通知 | |
| `notifyDeliveryTask` | 配送任務通知 | |
| `notifyLiftingTask` | 頂升任務通知 | T300 特有 |

### 1.4 硬件控制與感知

| API 名稱 | 中文說明 | 備註 |
|---------|--------|------|
| `notifyRobotPower` | 機器電量通知 | |
| `notifyElevatorUtilizeState` | 機器梯控通知 | 電梯聯動 |
| `notifyLiftingStatus` | 頂升機構狀態通知 | T300 特有 |
| `notifyRechargeStatus` | 回充狀態通知 | |

## 二、通用接口 (General APIs)

查詢類接口，用於獲取門店、地圖、機器人等基礎信息。

### 2.1 門店管理

| API 名稱 | 中文說明 |
|---------|--------|
| 獲取門店列表 | 查詢所有門店信息 |
| 門店綁定/解綁 | 機器人與門店的綁定關係管理 |
| 門店機器列表 | 查詢指定門店下的所有機器人 |

### 2.2 地圖信息

| API 名稱 | 中文說明 | 備註 |
|---------|--------|------|
| 門店下地圖列表 | 獲取門店所有地圖 | |
| 機器可用地圖列表 | 獲取機器人可使用的地圖 | |
| 機器當前使用地圖 | 查詢當前激活的地圖 | |
| 獲取地圖詳情 V2 | 地圖詳細信息 | V2 版本 |
| 繪製解析後的地圖 V2 | 渲染地圖圖像 | V2 版本 |
| 獲取機器實時地圖位置 | 機器人在地圖上的即時座標 | |
| 獲取機器當前使用地圖的點位 | 地圖中已定義的導航點 | |
| 獲取點位分組 | 點位的分組信息 | |

### 2.3 機器人信息

| API 名稱 | 中文說明 |
|---------|--------|
| 機器人通用狀態查詢 | 查詢機器人綜合狀態 |

### 2.4 機器人組

| API 名稱 | 中文說明 |
|---------|--------|
| 獲取綁定的機器人組 | 查詢機器人分組列表 |
| 獲取機器人組中的機器人 | 查詢指定分組內的機器人 |
